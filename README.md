# 🏐 VolleyBox

**VolleyBox** is a **2v2 volleyball game** in development, built with **Godot Engine 4.5** 🎮✨

Current focus:
- 🤖 **Single-player vs AI**
- 🧑‍🤝‍🧑 **Local play vs AI**
- 🖥️ **PC-only** (for now)

Planned for the future:
- 🌐 **Online multiplayer**

---

## 🛠️ Built With

- 🌱 [Godot Engine 4.5](https://godotengine.org/)

---

## 🚀 Getting Started (Run the Project)

### ✅ Requirements
- 🌱 Godot **4.6**
- 🧰 Git

### ▶️ Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Mcaldas11/VolleyBox.git
   ```
2. Open **Godot 4.5**
3. Click **Import** and select the project folder (or `project.godot`)
4. Press **Play** in the editor 🎬
